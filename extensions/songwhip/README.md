# Songwhip

Get a Songwhip link for the currently copied link of a song/album. Songwhip is a service that provides smart music links. These links show a list of platforms you can hear the song/album on.

## Supported platforms

- Spotify
- Apple Music
- YouTube
- Tidal
- Amazon Music
- YouTube Music
- Deezer
- Qobuz
- Napster
- iTunes
- Amazon
- And more...
